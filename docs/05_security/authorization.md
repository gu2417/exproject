# 권한 모델

## 1. 기능 권한

현재 버전은 모든 연결 클라이언트가 동일한 기능을 사용할 수 있다.

| 기능 | 권한 |
|------|------|
| 정류장/버스/지하철역 검색 | 모든 클라이언트 |
| 길찾기 최단 경로 | 모든 클라이언트 |
| 편의시설 조회 | 모든 클라이언트 |
| 운행 정보 조회 | 모든 클라이언트 |
| 즐겨찾기/최근 검색 | 클라이언트 로컬 |

## 2. 서버 파일 접근

클라이언트는 서버 파일 경로를 지정하거나 수정할 수 없다.
서버는 정해진 `data/`, `logs/` 경로만 사용한다.

## 3. 로컬 파일 접근

즐겨찾기와 최근 검색 기록은 클라이언트 실행 환경의 로컬 파일이다.
다른 클라이언트와 공유하지 않으며 서버 권한 모델에 포함하지 않는다.
