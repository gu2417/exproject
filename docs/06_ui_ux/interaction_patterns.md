# 상호작용 패턴

## 1. 입력

- 메뉴 번호는 숫자만 허용한다.
- 검색어는 앞뒤 공백을 제거한다.
- 빈 문자열은 서버에 보내지 않고 클라이언트에서 안내한다.

## 2. 결과 없음

결과 없음은 오류와 구분해 표시한다.

```text
검색 결과가 없습니다.
다른 키워드로 다시 검색해주세요.
```

## 3. 재시도

서버 연결 실패, 파일 읽기 실패, 입력 오류는 사용자가 다시 시도할 수 있게 한다.
단, 서버 데이터 로딩 실패처럼 치명적인 오류는 프로그램 종료를 허용한다.

## 4. 최근 검색 저장

정류장, 버스, 지하철역, 환승, 편의시설 검색 시 최근 검색 기록을 저장한다.
검색 실패까지 저장할지는 구현 정책으로 정하되, 문서와 코드가 일치해야 한다.
