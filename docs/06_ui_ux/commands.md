# 명령 및 메뉴

## 메인 메뉴

```text
===== 아산시 대중교통 정보 시스템 =====

1. 정류장 검색
2. 버스 번호 검색
3. 지하철역 검색
4. 길찾기(최단 경로)
5. 주변 편의시설 조회
6. 즐겨찾기 조회
7. 즐겨찾기 추가
8. 즐겨찾기 삭제
9. 최근 검색 기록
0. 종료
```

## 메뉴별 입력

| 번호 | 기능 | 입력 |
|------|------|------|
| 1 | 정류장 검색 | 정류장명 또는 키워드 |
| 2 | 버스 번호 검색 | 버스 번호 |
| 3 | 지하철역 검색 | 역명 또는 키워드 |
| 4 | 길찾기(최단 경로) | 탑승 위치, 도착 위치 |
| 5 | 주변 편의시설 조회 | 정류장/역명, 카테고리 |
| 6 | 즐겨찾기 조회 | 없음 |
| 7 | 즐겨찾기 추가 | 유형, ID 또는 번호, 표시 이름 |
| 8 | 즐겨찾기 삭제 | 삭제할 목록 번호 |
| 9 | 최근 검색 기록 | 없음 |
| 0 | 종료 | 없음 |

## 취소 규칙

세부 입력 중 `0`만 입력하면 메인 메뉴로 돌아가도록 구현할 수 있다.
단, 버스 번호 `0` 같은 실제 값과 혼동되지 않도록 안내 문구를 함께 출력한다.
