# 콘솔 디자인 토큰

## 1. 구분선

| 용도 | 문자열 |
|------|--------|
| 화면 제목 | `=====` |
| 섹션 구분 | `-----` |
| 결과 번호 | `[1]` |

## 2. 상태 문자열

| 의미 | 표기 |
|------|------|
| 성공 | `완료` |
| 결과 없음 | `검색 결과가 없습니다.` |
| 입력 오류 | `입력값을 다시 확인해주세요.` |
| 서버 오류 | `서버 처리 중 오류가 발생했습니다.` |

## 3. 길이 제한

| 출력 요소 | 권장 폭 |
|-----------|---------|
| 한 줄 안내문 | 80자 이내 |
| 목록 항목 이름 | 40자 이내 표시 후 줄바꿈 |
| 주소 | 길면 다음 줄 들여쓰기 |

## 4. 인코딩

콘솔 출력은 UTF-8을 기준으로 한다. Windows 환경에서는 한글이 깨질 수 있으므로 실행 전
콘솔 인코딩 설정을 안내할 수 있다.
