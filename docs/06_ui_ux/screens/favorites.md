# 즐겨찾기 화면

## 조회

```text
----- 즐겨찾기 목록 -----
[1] 정류장 | STOP001 | 온양온천역
[2] 버스   | 100     | 100번
[3] 지하철 | ST001   | 배방역
```

## 추가

```text
----- 즐겨찾기 추가 -----
유형 선택: 1. 정류장  2. 버스  3. 지하철역
유형 입력(번호 또는 한글):
ID 또는 번호 입력:
표시 이름 입력:
```

## 삭제

```text
----- 즐겨찾기 목록 -----
[1] 정류장 | STOP001 | 온양온천역
삭제할 번호 입력:
즐겨찾기를 삭제했습니다: 정류장 | STOP001 | 온양온천역
```

## 중복

```text
이미 저장된 즐겨찾기입니다.
```
