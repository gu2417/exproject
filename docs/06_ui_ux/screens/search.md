# 검색 입력 화면

## 정류장 검색

```text
----- 정류장 검색 -----
정류장명 또는 키워드 입력: 
```

## 버스 번호 검색

```text
----- 버스 번호 검색 -----
버스 번호 입력: 
```

## 지하철역 검색

```text
----- 지하철역 검색 -----
역명 또는 키워드 입력: 
```

## 길찾기

```text
----- 길찾기 최단 경로 -----
탑승 위치 입력:
도착 위치 입력:
```

## 주변 편의시설 조회

```text
----- 주변 편의시설 조회 -----
정류장/역명 입력:
카테고리 입력:
```
