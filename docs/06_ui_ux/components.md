# 콘솔 컴포넌트

## 1. Header

```text
===== 아산시 대중교통 정보 시스템 =====
```

프로그램명 또는 현재 화면명을 표시한다.

## 2. Prompt

```text
검색어를 입력하세요: 
```

사용자 입력을 받는 줄이다. 빈 입력은 허용하지 않는다.

## 3. Result List

```text
[1] 온양온천역 정류장
    ID: BUS_STOP_001
    경유 버스: 100, 101, 200
```

여러 결과는 번호를 붙여 보여준다.

## 4. Status Badge

상태는 대괄호로 표시한다.

```text
[운행 시작] [곧 도착] [운행 종료]
[여유] [보통] [혼잡]
```

## 5. Error Message

```text
오류: 검색 결과가 없습니다.
```

오류 원인을 한 줄로 명확히 표시하고 메뉴로 복귀한다.
