# 클라이언트 모듈

| 파일 | 책임 |
|------|------|
| `main.c` | 클라이언트 시작, 서버 접속, 메뉴 루프 |
| `menu.c/h` | 메뉴 출력 및 메뉴 번호 검증 |
| `client_net.c/h` | TCP 연결, 패킷 송수신 |
| `request_builder.c/h` | 사용자 입력을 요청 패킷으로 변환 |
| `result_viewer.c/h` | 응답 패킷을 사람이 읽기 쉬운 출력으로 변환 |
| `favorite.c/h` | 즐겨찾기 파일 처리 |
| `recent_search.c/h` | 최근 검색 기록 파일 처리 |

## 클라이언트 모듈 규칙

- 서버에 보내기 전에 빈 입력과 과도하게 긴 입력을 검사한다.
- 서버 오류 응답을 그대로 노출하지 않고 사용자 안내 문구로 변환한다.
- 로컬 파일 기능 실패는 서버 검색 기능을 막지 않는다.
