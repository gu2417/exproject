# 용어 정의

| 용어 | 정의 |
|------|------|
| 정류장 | 버스가 정차하는 위치. `bus_stops.csv`의 행으로 관리한다. |
| 버스 노선 | 버스 번호와 출발지, 종점, 배차 간격, 경유 정류장 순서를 포함하는 이동 경로. |
| 지하철역 | 지하철 노선의 역 정보. 환승 가능 노선 정보를 포함할 수 있다. |
| 편의시설 | 정류장 또는 역 주변의 시설. 카테고리와 주소를 가진다. |
| 노드 | 환승 그래프에서 정류장 또는 지하철역을 표현하는 정점. |
| 간선 | 그래프에서 두 노드 사이의 이동 가능 관계. 예상 소요시간을 가중치로 가진다. |
| 환승 경로 | 출발 노드에서 도착 노드까지 이동하기 위한 노드 목록과 환승 횟수. |
| 도착 예정시간 | 테스트 CSV의 시간대별 데이터를 기반으로 표시하는 예상 도착 분. |
| 혼잡도 | `여유`, `보통`, `혼잡` 등급으로 표현하는 시간대별 테스트 값. |
| 운행 상태 | `운행 시작`, `곧 도착`, `운행 종료` 중 하나로 표시하는 상태. |
| 즐겨찾기 | 사용자가 자주 찾는 정류장, 버스 번호, 지하철역을 로컬 파일에 저장한 항목. |
| 최근 검색 기록 | 사용자의 검색 유형, 키워드, 시간을 로컬 파일에 저장한 기록. |
| 패킷 | 클라이언트와 서버가 주고받는 텍스트 메시지. `<TYPE>|<PAYLOAD>\n` 형식을 따른다. |
