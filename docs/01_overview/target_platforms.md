# 대상 플랫폼

## 1. 지원 환경

| 항목 | 지원 기준 |
|------|-----------|
| 언어 | C11 |
| 운영체제 | Windows, Linux |
| 네트워크 | TCP/IP socket |
| 데이터 저장 | CSV 파일, 텍스트 로그, 텍스트 사용자 데이터 |
| UI | 콘솔 메뉴 |

## 2. 컴파일러

| 플랫폼 | 컴파일러 | 비고 |
|--------|----------|------|
| Linux | `gcc` | POSIX socket, `pthread` 사용 가능 |
| Windows | MinGW `gcc` | Winsock 초기화 필요 |

## 3. 포터빌리티 원칙

- 소켓 API 차이는 `src/common/net_compat.h` 계층에서 감춘다.
- 경로 구분자는 공통 유틸리티에서 처리하거나 상대 경로를 기준으로 사용한다.
- CSV, 로그, 프로토콜 문자열은 UTF-8을 기준으로 한다.
- Windows 콘솔 한글 출력 문제는 실행 안내 또는 콘솔 코드페이지 설정으로 보완한다.

## 4. 실행 제약

- 서버와 클라이언트는 같은 호스트 또는 같은 네트워크에서 실행하는 것을 기본으로 한다.
- 실시간 외부 API 호출은 수행하지 않는다.
- 시연 데이터는 `data/` 폴더의 CSV 파일과 테스트 CSV에 의존한다.
