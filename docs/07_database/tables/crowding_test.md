# crowding_test.csv

시간대별 혼잡도 테스트 데이터다.

| 컬럼 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `route_no` | string | Y | 버스 번호 또는 노선 번호 |
| `node_id` | string | Y | 정류장/역 ID |
| `time_slot` | string | Y | 시간대 |
| `crowding_level` | string | Y | 혼잡도 등급 |

## 혼잡도 등급

| 값 | 의미 |
|----|------|
| `여유` | 좌석 또는 공간 여유가 있음 |
| `보통` | 일반적인 이용 수준 |
| `혼잡` | 이용자가 많은 시간대 |
