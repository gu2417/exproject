# CSV/파일 데이터 실패 모드

DBMS는 사용하지 않지만, 파일 기반 데이터에서 발생할 수 있는 실패를 정의한다.

| 실패 모드 | 영향 | 처리 |
|-----------|------|------|
| CSV 파일 없음 | 서버 데이터 로딩 실패 | 파일명 로그 후 종료 |
| 필수 컬럼 누락 | 해당 데이터 사용 불가 | 컬럼명 로그 후 종료 또는 제외 |
| 빈 ID | 검색/참조 오류 | 행 건너뛰기, 로그 |
| 중복 ID | 잘못된 매칭 | 첫 항목 유지 또는 로딩 실패 |
| 잘못된 참조 | 경로/노선 출력 오류 | 참조 오류 로그, 해당 관계 제외 |
| 숫자 변환 실패 | 위치/시간 계산 오류 | 기본값 처리 또는 행 제외 |
| 인코딩 문제 | 한글 깨짐, 검색 실패 | UTF-8 저장 원칙 안내 |

## 로컬 파일 실패

| 파일 | 실패 시 처리 |
|------|--------------|
| `favorites.txt` | 빈 즐겨찾기로 처리 |
| `recent_search.txt` | 빈 기록으로 처리 |
| `server.log` | 콘솔 로그는 계속 출력 |
