# 엣지 케이스

## 검색

| 케이스 | 기대 동작 |
|--------|-----------|
| 같은 이름의 정류장이 여러 개 | ID와 지역을 함께 표시 |
| 검색어가 한 글자 | 결과가 많을 수 있으므로 목록 제한 가능 |
| 버스 번호에 공백 포함 | trim 후 검색 |
| 지하철역명 일부만 입력 | 부분 검색 허용 |
| 결과 없음 | `NOT_FOUND` 안내 |

## 환승

| 케이스 | 기대 동작 |
|--------|-----------|
| 출발지와 도착지가 같음 | 환승 0회, 소요시간 0분 또는 안내 |
| 출발지 중복 매칭 | 후보 목록 제공 또는 첫 번째 매칭 사용 정책 명시 |
| 경로 없음 | `NOT_FOUND` |
| 그래프 노드 누락 | 데이터 오류 로그 |

## 파일

| 케이스 | 기대 동작 |
|--------|-----------|
| 즐겨찾기 파일 없음 | 자동 생성 또는 빈 목록 |
| 최근 검색 파일 손상 | 읽을 수 있는 행만 표시 |
| 로그 폴더 없음 | 생성 시도, 실패 시 콘솔 출력 |

## 네트워크

| 케이스 | 기대 동작 |
|--------|-----------|
| 서버 실행 전 클라이언트 실행 | 연결 실패 안내 |
| 요청 중 서버 종료 | 연결 끊김 안내 |
| 패킷 중간에 연결 종료 | 세션 정리 |
