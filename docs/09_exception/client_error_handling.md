# 클라이언트 오류 처리

## 1. 입력 오류

| 상황 | 처리 |
|------|------|
| 메뉴 번호가 아님 | 오류 안내 후 메인 메뉴 |
| 범위 밖 메뉴 | 오류 안내 후 메인 메뉴 |
| 빈 검색어 | 서버 전송 없이 재입력 |
| 너무 긴 입력 | 길이 안내 후 재입력 |

## 2. 네트워크 오류

| 상황 | 처리 |
|------|------|
| 서버 연결 실패 | IP/포트 확인 안내 |
| 요청 전송 실패 | 연결 끊김 안내 후 종료 또는 재접속 |
| 응답 수신 실패 | 서버 상태 확인 안내 |

## 3. 로컬 파일 오류

즐겨찾기와 최근 검색 기록 파일 처리 실패는 치명적 오류가 아니다.
사용자에게 안내하고 메인 메뉴로 복귀한다.

```text
로컬 저장 파일을 읽을 수 없습니다.
검색 기능은 계속 사용할 수 있습니다.
```
