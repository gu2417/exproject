# 오류 분류

| 분류 | 코드 | 예 |
|------|------|----|
| 정상 | 0 | 검색 성공 |
| 결과 없음 | 1 | 키워드 매칭 없음 |
| 요청 오류 | 2 | 빈 검색어, 필드 부족, 알 수 없는 TYPE |
| 데이터 오류 | 3 | CSV 누락, 필수 컬럼 누락, 잘못된 참조 |
| 서버 오류 | 4 | 메모리 할당 실패, 예기치 않은 내부 오류 |

## 처리 원칙

- 사용자 입력 문제는 `INVALID_REQUEST`로 응답한다.
- 검색 결과가 없는 상황은 서버 오류가 아니다.
- CSV 로딩 실패는 서버 시작 단계에서 명확히 보고한다.
- 내부 오류는 로그를 남기고 가능한 경우 연결을 유지한다.
