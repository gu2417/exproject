# 사용자 편의 기능(FR-U01 ~ U06)

## FR-U01 즐겨찾기 추가 - P3

| 항목 | 내용 |
|------|------|
| 입력 | 유형, ID 또는 이름 |
| 저장 파일 | `client_data/favorites.txt` |
| 저장 형식 | `type|id|name` |
| 처리 위치 | 클라이언트 로컬 |

## FR-U02 즐겨찾기 조회 - P3

| 항목 | 내용 |
|------|------|
| 입력 | 없음 |
| 출력 | 저장된 즐겨찾기 목록 |
| 파일 없음 | 빈 목록으로 처리하고 필요 시 파일 생성 |

## FR-U03 즐겨찾기 중복 방지 - P3

| 항목 | 내용 |
|------|------|
| 중복 기준 | `type` + `id` |
| 처리 | 기존 항목이 있으면 추가하지 않고 안내 출력 |
| 성공 기준 | 같은 항목이 파일에 두 번 저장되지 않음 |

## FR-U04 즐겨찾기 삭제 - P3

| 항목 | 내용 |
|------|------|
| 입력 | 즐겨찾기 목록 번호 |
| 처리 | 선택한 번호의 항목을 제외하고 `favorites.txt` 재저장 |
| 예외 | 파일 없음 또는 없는 번호는 안내 메시지 출력 |

## FR-U05 최근 검색 기록 저장 - P3

| 항목 | 내용 |
|------|------|
| 저장 파일 | `client_data/recent_search.txt` |
| 저장 형식 | `type|keyword|timestamp` |
| 저장 시점 | 검색 요청 성공 또는 검색 시도 직후 |

## FR-U06 최근 검색 기록 조회 - P3

| 항목 | 내용 |
|------|------|
| 입력 | 없음 |
| 출력 | 검색 유형, 키워드, 시간 |
| 정렬 | 최신순 표시 권장 |
| 예외 | 파일 손상 시 읽을 수 없는 행은 건너뜀 |

## FR-U07 메뉴 기반 UI - P3

| 항목 | 내용 |
|------|------|
| UI | 콘솔 메뉴 |
| 메뉴 | 1~9번 기능, 0번 종료 |
| 검증 | 숫자 외 입력, 범위 밖 번호, 빈 검색어 처리 |
| 관련 문서 | [`06_ui_ux/`](../06_ui_ux/) |
