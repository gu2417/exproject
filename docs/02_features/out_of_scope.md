# 범위 제외 항목(FR-X01 ~ X05)

## 제외 목록

| ID | 제외 항목 | 제외 사유 |
|----|-----------|-----------|
| FR-X01 | 채팅 기능 | 프로젝트 주제 제한에 따라 구현하지 않는다. |
| FR-X02 | 학생정보관리 | 프로젝트 주제 제한에 따라 구현하지 않는다. |
| FR-X03 | 실시간 API 혼잡도 연동 | 시간대별 테스트 데이터로 대체한다. |
| FR-X04 | 정확한 실시간 위치 추적 | 정적 CSV와 테스트 데이터를 기반으로 안내한다. |
| FR-X05 | GUI 애플리케이션 | 콘솔 메뉴 기반 클라이언트로 범위를 제한한다. |

## 문서화 원칙

- 제외 항목은 메뉴와 패킷 명세에 포함하지 않는다.
- 실시간 정확도를 암시하는 표현을 피한다.
- 향후 확장 가능성은 언급할 수 있으나 현재 구현 범위와 구분한다.
