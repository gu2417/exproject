# 서버/클라이언트 및 로그(FR-N01 ~ N05)

## FR-N01 서버 실행 - P0

| 항목 | 내용 |
|------|------|
| 실행 파일 | `transit_server` |
| 입력 | 포트 번호 또는 기본 포트 |
| 처리 | socket 생성, bind, listen, accept loop |
| 실패 | 포트 사용 중, 권한 오류, 소켓 생성 실패 |

## FR-N02 클라이언트 접속 - P0

| 항목 | 내용 |
|------|------|
| 실행 파일 | `transit_client` |
| 입력 | 서버 IP, 포트 |
| 처리 | TCP connect 후 메뉴 표시 |
| 실패 | 연결 실패 메시지 출력 후 재시도 또는 종료 |

## FR-N03 요청 라우팅 - P0

| 항목 | 내용 |
|------|------|
| 모듈 | `router.c` |
| 입력 | 패킷 타입 |
| 처리 | 검색, 환승, 편의시설, 운행 정보 핸들러로 분기 |
| 예외 | 알 수 없는 타입은 `INVALID_REQUEST` |

## FR-N04 서버 로그 저장 - P3

| 항목 | 내용 |
|------|------|
| 파일 | `logs/server.log` |
| 내용 | 시간, 클라이언트 IP, 요청 타입, 처리 결과, 오류 |
| 동시성 | mutex 또는 단일 로그 함수로 기록 순서 보호 |
| 출력 | 파일 저장과 서버 콘솔 출력 병행 |

## FR-N05 연결 종료 처리 - P0

| 항목 | 내용 |
|------|------|
| 조건 | 클라이언트 종료, recv 0, 네트워크 오류, 메뉴 0번 |
| 처리 | 세션 비활성화, 소켓 close, 로그 기록 |
| 성공 기준 | 파일 디스크립터 누수와 서버 크래시가 없음 |
